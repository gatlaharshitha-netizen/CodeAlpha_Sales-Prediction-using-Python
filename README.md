# Sales Prediction using Python

## Description
This project predicts product sales using Machine Learning by analyzing advertising data. It explores the relationship between advertising expenditure on different media (such as TV, Radio, and Newspaper) and sales. The project performs data preprocessing, exploratory data analysis (EDA), visualization, model training, and evaluation to accurately predict future sales.

## Features
- Load and analyze the sales dataset
- Data preprocessing and cleaning
- Check for missing values
- Exploratory Data Analysis (EDA)
- Visualize relationships between advertising channels and sales
- Train a Machine Learning model
- Predict sales based on advertising expenditure
- Evaluate model performance using regression metrics

## Dataset
The dataset contains advertising expenditure and corresponding sales values.

### Features
- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget

### Target Variable
- Sales

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Algorithm
**Linear Regression**

## Project Workflow
1. Import the required Python libraries.
2. Load the sales dataset.
3. Explore the dataset and check for missing values.
4. Perform data visualization.
5. Split the dataset into training and testing sets.
6. Train the Linear Regression model.
7. Predict sales using the trained model.
8. Evaluate the model using regression metrics.
9. Display the prediction results.

## How to Run
1. Install the required Python libraries:
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn
   ```
2. Place the dataset (`advertising.csv` or your dataset file) in the project folder.
3. Run the Python script or Jupyter Notebook.
4. View the visualizations, model performance, and predicted sales.

## Output
- Dataset overview
- Missing value analysis
- Data visualizations
- Correlation analysis
- Predicted sales values
- Model evaluation metrics such as:
  - Mean Absolute Error (MAE)
  - Mean Squared Error (MSE)
  - Root Mean Squared Error (RMSE)
  - R² Score
